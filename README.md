# Hiroshima AI Assurance Council Website

Static public website package for the Hiroshima AI Assurance Council.

## Pages

- `index.html` - Home and technical evidence overview
- `about.html` - Council structure and operating scope
- `protocol.html` - HAAP non-normative overview / discussion draft v0.1
- `projects.html` - Phase 1 PoC scope and next-phase targets
- `beacon-cycle.html` - Beacon Cycle concept and diagram
- `contact.html` - Official contact entry point
- `404.html` - Not-found page

## Publication Notes

- The Beacon Cycle diagram uses `beacon-cycle-en.svg`.
- `robots.txt` is included without a sitemap URL because the production domain is not fixed in this package.
- External technical evidence links point to public GitHub repositories operated by GhostDrift.
